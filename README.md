# Movie Rating Tracker

A simple Python program that lets you track movies you've watched and rate them.

## What it does:
- Add movies you've watched
- Rate each movie
- Stores all your movies and ratings
- Keeps adding movies until you say stop

## How to run:
1. Make sure you have Python installed
2. Run `python movie-rating-tracker.py`
3. Enter a movie name
4. Rate it
5. Choose to add another or stop

## Example:
```
Enter movie: Inception
What would you rate this movie: 9
Add another movie? Y/N Y
Enter movie: Interstellar
What would you rate this movie: 10
Add another movie? Y/N N
```
